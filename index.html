How the Web Works

Hypertext Markup Language, a standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects on World Wide Web pages. You can find more information on HTML here on thewiki page.

Tags and Elements

HTML documents are made of HTML elements. When writing HTML, we tell browsers the type of each element by using HTML tags. This video explains the distinction well.

Thinking like a developer

The following are good ways of training yourself to think the same way as aweb developer:

  • Procedural thinking (One big problem into step 1, 2 , 3 etc)
  • Abstract thinking (1 + 1 = 2)
  • System Debugging (Problem; possible causes; possible solutions)
  • Tech Empathy (how a cpu thinks)
  • Systems thinking (Big issues into Component pieces)

Inline vs Block Elements

HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them.
